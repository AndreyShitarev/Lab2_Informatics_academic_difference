# Отчёт по выполнения лабораторной работы №2

## Задача
Написать .bash скрипт, который будет принимать IPv4-адрес в десятичной форме и выводить его в двоичной форме

## Для его получения с совершил следующие шаги:
### 1. Создал директорию для этой лабораторной с помошью команды `mkdir lab2` 
### 2. Перешёл в выбранную директорию при помощи команды `cd lab2`
### 3. Находясь в директории `lab2`, при помощи команды `> IP.bash` создал файл для написания скрипта
### 4. Командой 'nano IP.bash' открыл файл-скрип через текстовый редактор `nano`
### 5. При помощи команды `sudo apt install bc' установил утилиту `bc`, которая выполняет функции консольного калькулятора
### 6. Далее я работал в файле `IP.bash'. Указал путь к bash интерпритатору при помощи `#!/usr/bin/env bash`
### 7. `IPVFOUR="$1"` инициализировал входную переменную
### 8. `IPBINARY=""` инициализировал вызодную переменную (туда пише результат)
### 9. `BEGINNER=""` ввёл переменную-разделитель, вообще она равна `.`, но, чтобы избежать точки в начале вывода, сначала она пустая
### 10. `IFS='.' read -r -a PARTS <<< "$IPVFOUR"` разделил поля и создал список `PARTS` для разделённых частей
### 11. `for PART in "${PARTS[@]}"; do` начал цикл для каждого элемента `PART` списка `PARTS`
### 12. `BINPART=$(echo "obase=2;$PART" | bc)` написал команду, переводящую каждую часть в двоичную форму
### 13. `OCT=$(printf "%08d" "$BINPART")` написал команду, которая заполняет строку ведущими нулями для правильного двоичного представления
### 14. `IPBINARY="$IPBINARY$BEGINNER$OCT"` написал команду, собирающую конечный IP адрес (добавляет элемент к общему результату и, начианая со второй позиции делит точкой)
### 15. `BEGINNER="."` устанавливаем в качестве разделителя `.` 
### 16. `done` завершаю цикл
### 17. `echo "$IPBINARY"` вывожу результат работы скрипта в терминал
